### рҹҢҹ Features in v1.0.4 (бһҳбһ»бһҒбһ„бһ¶бһҡбһҗбҹ’бһҳбһёбҹ—)бҹ–
- **AI Khmer Dubbing**: Auto-transcribe audio and dub videos into fluent Khmer voiceover. (бһ”бһүбҹ’бһ…бһјбһӣбһҹбҹҶбһЎбҹҒбһ„бһҮбһ¶бһ—бһ¶бһҹбһ¶бһҒбҹ’бһҳбҹӮбһҡбһҠбҹ„бһҷбһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·бһҸбһ¶бһҳбһ”бһ…бҹ’бһ…бҹҒбһҖбһңбһ·бһ‘бҹ’бһҷбһ¶ AI)
- **DramaBox Scraper**: Crawl drama catalog and download HLS segments into MP4. (бһ‘бһ¶бһүбһҷбһҖбһңбһёбһҠбҹҒбһўбһјбһ—бһ¶бһӮбһ–бһё DramaBox бһ“бһ·бһ„бһ”бһүбҹ’бһ…бһјбһӣбһҡбһҪбһҳбһӮбҹ’бһ“бһ¶бһҮбһ¶ MP4 бһҹбҹ’бһңбҹҗбһҷбһ”бҹ’бһҡбһңбһҸбҹ’бһҸбһ·)
- **ReelShort Scraper**: Fetch and download high-quality MP4 videos. (бһ‘бһ¶бһүбһҷбһҖбһңбһёбһҠбҹҒбһўбһјбһ—бһ¶бһӮбһӣбһҝбһ“бһҡбһ бҹҗбһҹбһ–бһё ReelShort)
- **Watermark Blurring**: Blur or crop logos/watermarks cleanly. (бһ”бһ·бһ‘бһ”бһ¶бҹҶбһ„ Logo бһ¬ Watermark бһӣбһҫбһңбһёбһҠбҹҒбһўбһј)
- **Persistent Favorites**: Save and reload catalog items easily. (бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһҡбһҝбһ„бһҠбҹӮбһӣбһ…бһјбһӣбһ…бһ·бһҸбҹ’бһҸ бһ“бһ·бһ„бһ‘бһ¶бһүбһҷбһҖбһ‘бһ·бһ“бҹ’бһ“бһ“бҹҗбһҷбһЎбһҫбһ„бһңбһ·бһүбһҡбһ бҹҗбһҹ)
- **Direct Gallery Sync**: Save downloads directly to system gallery. (бһҡбһҖбҹ’бһҹбһ¶бһ‘бһ»бһҖбһңбһёбһҠбҹҒбһўбһјбһҠбҹӮбһӣбһ”бһ¶бһ“бһ‘бһ¶бһүбһҷбһҖбһҡбһҪбһ… бһ‘бҹ…бһҖбҹ’бһ“бһ»бһ„ Gallery бһ‘бһјбһҡбһҹбҹҗбһ–бҹ’бһ‘бһ—бҹ’бһӣбһ¶бһҳбҹ—)
